# Cashback-Optimizer Popup

Ein kompaktes Userscript, das dir automatisch anzeigt, ob es für den aktuellen Onlineshop Cashback oder Gutscheine gibt.

## Features
- 🔍 **Direktsuche:** Ein Klick öffnet die passenden Ergebnisse auf Cashback-Optimizer.de.
- 🚀 **Automatische Erkennung:** Erkennt hunderte Shops basierend auf der URL.
- 🔗 **Verlinkung:** Verlinkt direkt auf die spezifische Cashback-Unterseite oder nutzt optimierte DuckDuckGo Lucky Searches (!ducky) für maximale Trefferquoten.
- ⚡ **Performance:** Nutzt lokales Caching, um keine unnötigen Netzwerkanfragen zu stellen.
- 📱 **Mobile Optimized:** Spezielle UI-Anpassungen für iPhone/Android inkl. Safe-Area-Support und optimierter Schriftgröße (15px).

## Installation
1. Installiere die Browser-Erweiterung **Tampermonkey** oder **Violentmonkey**.
2. Klicke auf den folgenden Link: [👉 JETZT INSTALLIEREN](https://github.com/ysamjo/cashback-optimizer-popup/raw/refs/heads/main/cashback-optimizer.user.js)
3. Bestätige die Installation in deinem Browser.

Hinweis:
Dieses Skript ist ein Community-Projekt und nutzt die API/Daten von cashback-optimizer.de.
