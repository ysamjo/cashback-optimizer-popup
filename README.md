# Cashback-Optimizer Popup

Ein kompaktes Userscript, das dir automatisch anzeigt, ob es für den aktuellen Onlineshop Cashback oder Gutscheine gibt.

## Features
- 🚀 **Automatische Erkennung:** Erkennt hunderte Shops basierend auf der URL.
- 🔍 **Direktsuche:** Ein Klick öffnet die passenden Ergebnisse ohne Cashback-Optimizer.de aufrufen zu müssen.
- 🔗 **Verlinkung:** Verlinkt direkt auf die spezifische Cashback-Unterseite oder nutzt optimierte DuckDuckGo Lucky Searches für Deeplinks. #savedyouaclick
- ⚡ **Performance:** Nutzt lokales Caching, um keine unnötigen Netzwerkanfragen zu stellen.
- 📱 **Mobile Optimized:** Spezielle UI-Anpassungen für iPhone/Android.

## Installation
1.
   - Desktop: Installiere die Browser-Erweiterung [**Tampermonkey**](https://www.tampermonkey.net/).
   - iOS: Installiere die [**Userscripts App**](https://apps.apple.com/us/app/userscripts/id1463298887).
2. Klicke auf den folgenden Link: [👉 JETZT INSTALLIEREN](https://github.com/ysamjo/cashback-optimizer-popup/raw/refs/heads/main/cashback-optimizer.user.js)
3. Bestätige die Installation in deinem Browser.

Hinweis:
Dieses Skript ist ein Community-Projekt und nutzt die Daten von cashback-optimizer.de.
